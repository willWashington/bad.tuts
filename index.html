About bad.tuts

bad.tuts is a tutorial website created for software noobs by a beginner. Test

I am learning to develop software. I've found throughout my life that I learn better when I teach, so I created this website to teach others as I learn. The hope here is that I learn more by maintaining a project while also teaching other beginners how to build software.

Using this Website

The layout for this website will be exactly what you see here. All of the tutorials will be in a series of cards like this one. Eventually, the nav bar will be full of language sites. On each language site will be a series of cards like these that describe how to manipulate languages in specific ways to accomplish goals.

Eventually, I want to add a Wiki Mode button that will swap the entire layout for this website into a wiki format.

Please choose a language from the nav bar at the top of the page to begin exploring!
